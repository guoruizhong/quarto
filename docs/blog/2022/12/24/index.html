Setup SSH Remote Keys

Login to server without using password with SSH remote key

  1. Add the host into the hosts folder
  • Mac: code /etc/hosts
  • Windows: C:\Windows\System32\drivers\etc\hosts
### On the local machine: 
172.20.***.** nucleus.ccpa nucleus
172.20.***.** lipid-droplets.ccpa lipid-droplets
  1. Generate public key
### Windows
local> cd ~/.ssh
local> ssh-keygen -t ed25519
local> ls
local> more id_ed25519.pub 
### Mac
local> cd ~/.ssh
local> ssh-keygen -t ed25519
local> ls
local> cat id_ed25519.pub
  1. Add the public key to remote server
###  On the remote server
remote> cd ~/.ssh
remote> chown -R chase:users .ssh
remote> chmod 700 .ssh
remote> scp username@local://home/user/.ssh/id_ed25519.pub .ssh/authorized_keys
remote> chmod 644 .ssh/authorized_keys
### Or on local machine
### Windows
# local> scp id_ed25519.pub remote:/home/user/.ssh/authorized_keys

### Mac
### Also copy the key to authorized_keys in the remote sever
local> cat id_ed25519.pub

Adding colors to Bash scripts

In the remote server, add the following script to the ~/.bashrc

PS1='\[\033[1;36m\]\u\[\033[1;37m\] @\[\033[1;32m\] \h:\[\033[1;33m\] \w\[\033[0;97m [\t]\[\033[0;97m \n\[\033[1;35m\]\$\[\033[0m\] '

Supercharge Terminal

### Install [Home-brew](https://brew.sh/)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

### Change default shell from bash to zsh
chsh -s $(which zsh)

### Install oh-my-zsh in the terminal
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

###  change default theme to ys
open ~/.zshrc

ZSH_THEME="ys"

### install oh-my-zsh plugins
git clone https://github.com/zsh-users/zsh-syntax-highlighting.git ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-syntax-highlighting

git clone https://github.com/zsh-users/zsh-autosuggestions ${ZSH_CUSTOM:-~/.oh-my-zsh/custom}/plugins/zsh-autosuggestions


### make oh-my-zsh plugins working
open ~/.zshrc

plugins=(
  git
  zsh-autosuggestions
  zsh-syntax-highlighting
  colored-man-pages
  colorize
)

### Make it working
source ~/.zshrc

Mount a remote files

  1. Install macFUSE and SSHFS: https://osxfuse.github.io/

  2. Create mount directory in local

$ mkdir ~/mount/mountpoint
  1. Mount directory
sshfs «user»@«host»: «mountpoint»
  1. Un-mounting the remote volume
$ umount ~/mount/mountpoint

VScode SSH using Keys

open .ssh/config, add the following

Host nucleus
    HostName 172.20.***.** ## host ip address
    User usr_name
    IdentityFile "Users/usr_name/.ssh/id_******" ### private key
