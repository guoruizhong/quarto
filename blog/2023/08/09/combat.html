Adjusting for batch effect with {Combat}

batch <- pheno$batch
modcombat <- model.matrix(~1, data = pheno)
modcancer <- model.matrix(~cancer, data = pheno)
combat_edata <- ComBat(
    dat = edata, 
    batch = batch,
    mod = modcombat, 
    par.prior =  TRUE,
    prior.plots = FALSE
    )

Comparing the combat and linear adjustment
